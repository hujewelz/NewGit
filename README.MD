###插入代码片段
我们需要在代码的上一行和下一行用··· 标记。``` 不是三个单引号，而是数字1左边，Tab键上面的键。要实现语法高亮那么只要在 ``` 之后加上你的编程语言即可（忽略大小写）。c++语言可以写成c++也可以是cpp。
```Java
public static void main(String[] args) {} //java
```
```c
int main(int args, char *argv[]) {} //C
```
```Bash
echo "hello github" #Bash
```
